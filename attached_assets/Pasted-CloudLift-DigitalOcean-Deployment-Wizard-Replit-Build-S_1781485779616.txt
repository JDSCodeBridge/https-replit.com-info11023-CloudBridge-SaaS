CloudLift DigitalOcean Deployment Wizard - Replit
Build Specification
Goal
Improve the CloudLift deployment flow and make it usable by complete beginners with no cloud or
engineering experience. The user should never need to understand infrastructure, servers,
containers, or DevOps concepts.
Step 1: Cloud Provider Selection
Create a new screen after AI Analysis.
Display four cloud providers: DigitalOcean (Recommended) AWS Azure Google Cloud For each
provider show: Beginner Friendliness Estimated Monthly Cost Deployment Speed Recommended
Use Case Highlight DigitalOcean as: "Best for most founders and AI applications."
■ Recommended
Estimated Cost: $5-$25/month
Difficulty: Easy

Design Requirements
Keep existing CloudLift design language: Dark theme Neon blue accents Large buttons
Step-by-step wizard Make every screen understandable to someone who has never used cloud
hosting before. Avoid technical jargon whenever possible. Explain every action in plain English.
Focus on DigitalOcean first before AWS, Azure, or Google Cloud.
