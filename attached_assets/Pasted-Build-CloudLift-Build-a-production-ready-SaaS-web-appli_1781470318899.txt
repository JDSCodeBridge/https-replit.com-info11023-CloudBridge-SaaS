Build CloudLift

Build a production-ready SaaS web application called CloudLift.

Vision

CloudLift helps non-technical users launch AI-built applications to production.

Users build apps using Replit, Claude Code, Cursor, Lovable, Bolt, GitHub, or other AI coding tools.

CloudLift analyzes their repository and helps them deploy to cloud providers and app stores.

The platform should feel like “TurboTax for app deployment.”

⸻

Core User Flow

Step 1: User Registration

Users can:

* Create account
* Login
* Reset password

Authentication:

* Email/password
* Google OAuth
* GitHub OAuth

⸻

Step 2: Connect Repository

Users can:

* Connect GitHub account
* Select repository

Display:

* Repository name
* Framework detected
* Last update
* Deployment status

⸻

Step 3: AI Repository Analysis

Scan repository and detect:

Frontend:

* React
* Next.js
* Vue
* Angular
* Svelte

Backend:

* Node.js
* Python
* Flask
* Django
* FastAPI
* Express

Database:

* PostgreSQL
* MySQL
* MongoDB
* Supabase

Output:

Deployment Readiness Score

Example:

* Infrastructure: 90%
* Security: 80%
* Environment Variables: Missing
* Database Configuration: Complete

Generate recommendations.
