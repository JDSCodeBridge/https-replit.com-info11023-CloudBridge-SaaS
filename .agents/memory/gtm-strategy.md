---
name: CloudLift go-to-market strategy
description: User-confirmed product strategy for early revenue
---

**Strategy**: Concierge-first. Don't automate cloud deployments before finding patterns.

Steps to revenue:
1. Connect GitHub
2. Get AI analysis + readiness score
3. Pay $149 (Launch package)
4. Submit deployment request form
5. Owner manually handles the deployment

**Why:** Fastest path to revenue. Learn what customers actually need before automating. Automation comes after seeing patterns in real requests.

**How to apply:** Don't prioritize AWS/Azure/GCP automation features. Prioritize GitHub integration, AI analysis quality, payment flow, and service request workflow.
